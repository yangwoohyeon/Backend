<h2 align="center">봉사자와 도움이 필요한 이웃을 연결해주는 서비스 , Halpme(할프미)</h2>

**Halpme(할프미)**는 지역 내 청년 및 봉사자들과 거동이 불편한 이웃 또는 도움이 필요한 고령자들을 연결해줌으로써,
일상생활에서 발생하는 작은 어려움부터 긴급한 도움까지 빠르고 안전하게 지원받을 수 있도록 돕는 지역 기반 상호지원 서비스입니다.

<br>

<br>

## ✔️ 주요 기능

### 1. 도움 요청 게시판
- 거동이 불편한 이웃 또는 도움이 필요한 주민들이 도움 요청 글(심부름, 동행, 간단한 노동 등)을 쉽게 등록할 수 있습니다.
- 요청 지역(동 단위) 기반으로 게시글이 분류되어, 지역내 봉사자가 빠르게 확인할 수 있습니다.

### 2. 봉사자 매칭
- 봉사자는 자신의 위치와 가능한 시간대를 기준으로 주변의 도움 요청을 확인하고 참여할 수 있습니다.

### 3. 1:1 실시간 채팅
- 매칭이 완료되면 요청자와 봉사자는 실시간 채팅으로 상세 위치, 요청 내용, 준비물 등을 바로 조율할 수 있습니다.
- 이미지 전송 기능을 통해 장소나 상황을 더욱 명확하게 전달할 수 있습니다.


<br>

## 🎯 목표
- 소외되는 사회적 약자에게 도움이 필요한 순간 즉시 지원받을 수 있는 환경을 제공하는 것
- 지역 청년 및 봉사자들이 자연스럽게 사회 공헌 활동에 참여할 수 있는 구조를 만드는 것
- 동네 기반의 상호 연결을 통해 지역 공동체의 따뜻한 돌봄 문화를 확산하는 것
- 기술을 활용해 고립·단절 문제를 완화하고, 누구나 쉽게 도움을 주고받는 사회를 만드는 것



<br>

## 🛠️ 기술 스택
- **Backend**: Spring Boot, JPA, MySQL  
- **Frontend**: React, TypeScript
- **Infra/DevOps**: AWS EC2, AWS S3, GitHub Actions, Docker
